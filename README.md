# kgu_babmat
### Project
* 경기대학교 내의 식당 메뉴에 대해 리뷰를 작성하는 사이트
* 기간: 2023.01.08 ~ 2023.01.31(3.5주)
<br><br>

### Description
교내 식당의 위치와 이용시간, 메뉴, 가격 등을 제공하여 학생들에게 편의를 제공한다.
리뷰 작성을 통해 학우들의 불만족스러운 식사 및 불필요한 소비를 막아보자.
