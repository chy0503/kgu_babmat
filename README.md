# kgu_babmat
### Project
* 경기대학교 내의 식당 메뉴에 대해 리뷰를 작성하는 사이트
* 기간: 2023.01.08 ~ 2023.03.20(10주)
<br><br>

### Description
교내 식당의 위치와 이용시간, 메뉴, 가격 등을 제공하여 학생들에게 편의를 제공한다.
<br><br>

### Member
**박소현** - 노션 관리자
<br>
**이정은** - 지라 관리자
<br>
**조희연** - 깃허브 관리자
<br><br>

### Environment
- Language : Java, JavaScript, HTML, CSS
- DB : Firestore
- Framework : Spring Boot
- IDE : IntelliJ
- etc : oauth2, KakaoMap
<br><br>
